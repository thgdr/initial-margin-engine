# Initial Margin Engine

## Overview
This project implements a simplified version of a SIMM-style initial margin model using sensitivity-based risk aggregation.

## Features
- Delta sensitivity calculation
- Risk bucketing by maturity
- Correlation-based aggregation
- Portfolio-level IM calculation
